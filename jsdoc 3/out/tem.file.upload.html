<static> archiveAndUploadFile

Not available in Web Reports Allows you to archive a file or folder, writing it to disk, and then upload it. Streams data directly through ArchiveWriter into an HTTP request. 
 var result = tem.file.upload.archiveAndUploadFile(inputPath, outputPath, recurseFlag);
 var uploadInfo = result.split("\t");
 var url = uploadInfo[0];
 var size = uploadInfo[1];
 var sha1 = uploadInfo[2];

recurseFlag Boolean If true, will cause the archiver to add all subfolders to the archive (if the inputPath is a folder). Otherwise, the archive will contain only the input folder's contents, and not the contents of any subfolders.
