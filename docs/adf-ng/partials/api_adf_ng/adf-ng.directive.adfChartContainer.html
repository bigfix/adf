adfChartContainer (directive in module adf-ng )

Description

Chart container contains a chart item. It adds chart title and resizes its chart component when needed.

Usage

as attribute
<ANY adf-chart-container
     header="{String}"
     [resizeFn="{Function}"]>
   ...
</ANY>

Parameters

  • header – {String} –

    Chart header

  • resizeFn(optional) – {Function=} –

    Returns a function that can be called to force chart resize
