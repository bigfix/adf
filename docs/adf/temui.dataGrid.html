Class: dataGrid

temui. dataGrid

new dataGrid(data, columns, gridOptions)

dataGrid is the standard UI data grid component. It extends the Slick.Grid component from the SlickGrid jQuery library. Included with the ADF under libs is the version of the SlickGrid library used by the ADF along with its sample code. Please refer to its online/included documentation for details about the Slick.Grid component on which dataGrid based.
