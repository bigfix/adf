Class: Library

tem.library. Library

Not available in Web Reports General class used to to install library files. By default it installs into the folder defined by the user's environment variable 'TEMP'. The files are placed in the TEMP folder under:

(site name with spaces replace by underscores when propagated as part of site)(dashboard id with '.ojo' stripped off)/libraryName/libraryVersion

It requires a zip of the library files and an executable for unzipping the files.

If an existing library exists with the same name and version, then it aborts the install. If any files are located under the same libraryName but a different libraryVersion (basically a different version of the same library exists), then those files are deleted.

new Library(libraryName, libraryVersion, libraryZipName, zipExecutableName)

Parameters:
Name Type Argument Default Description
libraryName String Used to uniquely create and locate the library file in the common temp folder.
libraryVersion String Used to uniquely create and locate the library file within a specific library in the common temp folder.
libraryZipName String Name of zip file included in site that contains library files.
zipExecutableName String <optional>
 BFArchive.exe Name of zip executable included in site used to unzip the contents.
